# SEO Content Writer Skill

## Purpose
Create search-optimized content that ranks well, engages users, and demonstrates E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness).

## Activation
Use when you need to:
- Write new SEO-optimized content
- Rewrite existing content for better rankings
- Create content briefs for writers
- Optimize content for specific keywords
- Create Arabic content for MENA markets

## Content Types

### Service/Product Pages
```
Structure:
1. Clear H1 with primary keyword
2. Above-fold value proposition
3. Key benefits (bullet points)
4. Detailed service description
5. Process/How it works
6. Credentials/Trust signals
7. FAQ section
8. Clear CTA
9. Related services internal links
```

### Blog/Educational Content
```
Structure:
1. Engaging H1 (keyword + benefit)
2. Hook paragraph (address pain point)
3. Table of contents (for 2000+ words)
4. Logical H2/H3 structure
5. Multimedia (images, videos)
6. Expert quotes or data citations
7. Actionable takeaways
8. Related content links
9. Author bio (E-E-A-T)
```

## SEO Content Checklist

### Title/H1 Optimization
- [ ] Primary keyword in H1
- [ ] H1 appears only once
- [ ] H1 is engaging, not just keyword-stuffed
- [ ] Under 60 characters (for display)
- [ ] Matches search intent

### Meta Description
- [ ] 150-160 characters
- [ ] Includes primary keyword
- [ ] Includes call-to-action
- [ ] Unique for this page
- [ ] Compelling to click

### Content Structure
- [ ] Logical heading hierarchy (H1 â†’ H2 â†’ H3)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bullet points and numbered lists
- [ ] One idea per paragraph
- [ ] Transition sentences between sections

### Keyword Optimization
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in H1, one H2
- [ ] Secondary keywords naturally distributed
- [ ] Keyword density 1-2% (not forced)
- [ ] LSI/related terms included
- [ ] No keyword stuffing

### E-E-A-T Signals
- [ ] Author byline with credentials
- [ ] "Written by" or "Reviewed by" expert
- [ ] First-hand experience demonstrated
- [ ] Citations to authoritative sources
- [ ] Updated date displayed
- [ ] About/credentials page linked

### User Experience
- [ ] Answers search intent clearly
- [ ] Skimmable (headers, lists, bold)
- [ ] Images with alt text
- [ ] Internal links to related content
- [ ] External links to authoritative sources
- [ ] Mobile-friendly formatting
- [ ] Fast-loading (no huge images)

### Arabic Content Specific
- [ ] RTL formatting correct
- [ ] Dialect appropriate for audience
- [ ] Cultural relevance
- [ ] Arabic keywords naturally placed
- [ ] Mixed Arabic/English terms handled properly

## Content Quality Markers

### Must Have
- Answers the search query directly
- Provides unique value not found elsewhere
- Demonstrates expertise through detail
- Includes actionable information
- Has clear, logical structure

### Should Have
- Original images or custom graphics
- Video content (embedded)
- Data, statistics, or research citations
- Real examples or case studies
- Expert opinions or quotes

### Nice to Have
- Interactive elements
- Downloadable resources
- Infographics
- User-generated content (reviews, comments)
- Updated regularly

## Arabic Writing Best Practices

1. **Avoid literal translation**: Create native Arabic content
2. **Use appropriate dialect**: MSA for formal, colloquial for engagement
3. **Mixed terminology**: Keep technical terms in English if common
4. **Cultural references**: Include relevant local context
5. **Font choice**: Use Arabic-optimized fonts (Cairo, Tajawal, etc.)
6. **Reading flow**: RTL with proper punctuation
7. **Keyword balance**: Include both Arabic and transliterated searches
