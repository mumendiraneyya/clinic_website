# /seo:local Command

## Description
Perform a local SEO audit focused on Jordan and MENA markets, including Google Business Profile and directory optimization.

## Usage
```
/seo:local [URL or business name]
/seo:local https://abuobaydatajjarrah.com
/seo:local "عيادة أبو عبيدة الجراح"
```

## What It Does

1. Checks Google Business Profile status
2. Audits local directory presence
3. Verifies NAP consistency
4. Analyzes local schema markup
5. Reviews local content strategy
6. Assesses review profile
