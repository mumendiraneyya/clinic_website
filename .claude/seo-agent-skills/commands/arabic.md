# /seo:arabic Command

## Description
Audit and optimize a website for Arabic SEO, including RTL implementation, dialect considerations, and cultural factors.

## Usage
```
/seo:arabic [URL]
/seo:arabic https://abuobaydatajjarrah.com
```

## What It Does

1. Checks RTL implementation
2. Analyzes Arabic content quality
3. Reviews dialect appropriateness
4. Evaluates cultural considerations
5. Audits Arabic keyword optimization
