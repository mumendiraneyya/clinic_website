# /seo:keywords Command

## Description
Perform keyword research for a website or topic, with special attention to Arabic keywords and dialect variations.

## Usage
```
/seo:keywords [topic/URL] [region]
/seo:keywords "general surgery" jordan
/seo:keywords https://abuobaydatajjarrah.com
/seo:keywords "جراحة الفتق" mena
```

## What It Does

1. Identifies seed keywords from topic/URL
2. Expands to related keywords
3. Analyzes Arabic dialect variations
4. Groups by intent and topic cluster
5. Prioritizes by opportunity
