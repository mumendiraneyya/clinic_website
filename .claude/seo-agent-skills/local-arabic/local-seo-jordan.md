# Local SEO for Jordan Skill

## Purpose
Optimize websites for local search visibility in Jordan, focusing on Amman and other Jordanian cities, with attention to local directories, Google Business Profile, and regional search behavior.

## Activation
Use when you need to:
- Improve local search visibility in Jordan
- Optimize Google Business Profile for Jordanian businesses
- Get listed in Jordanian directories
- Target specific cities in Jordan
- Build local citations and reviews

## Google Business Profile Optimization

### Setting Up for Jordan

```markdown
1. Create/Claim Profile
   - Use business Arabic name + English
   - Example: "عيادة الدكتور معمن | Dr. Mu'men Clinic"

2. Categories
   - Primary: Most specific category
   - Additional: Up to 9 related categories
   - Example: "جراح عام" (General Surgeon)

3. Service Area
   - City: Amman (عمان)
   - If serving multiple areas, list all
```

## Reviews Strategy

### Getting Reviews in Jordan

```markdown
Approaches that work:

1. Direct Request
   - Ask satisfied patients directly
   - Provide simple instructions
   - Send WhatsApp with Google review link

2. Follow-up Message
   "شكراً لزيارتكم لعيادتنا. إذا كانت تجربتكم إيجابية،
   نرجو تقييمنا على جوجل: [رابط]"

3. QR Code in Clinic
   - Display QR linking to review page
   - At checkout/reception

4. Thank You Card
   - Physical card with review request
   - Given after successful procedure
```

### Review Response Templates (Arabic)

```markdown
Positive Review Response:
"شكراً جزيلاً [الاسم] على تقييمكم الكريم. سعدنا بخدمتكم
ونتمنى لكم دوام الصحة والعافية. نتطلع لرؤيتكم مجدداً."

Negative Review Response:
"شكراً [الاسم] على ملاحظاتكم. نأسف لأن تجربتكم لم تكن
كما تستحقون. نرجو التواصل معنا مباشرة على [الهاتف] لمناقشة
الأمر وإيجاد حل مناسب. رضاكم أولويتنا."
```
