# SERP Analysis Skill

## Purpose
Analyze search engine results pages and AI-generated answers to understand ranking patterns, SERP features, and AI citation behavior for target keywords.

## Activation
Use when you need to:
- Understand what Google displays for target keywords
- Identify SERP feature opportunities
- Analyze AI Overviews and AI answers
- Understand user intent from SERP layout
- Find featured snippet opportunities

## Process

### Step 1: SERP Feature Identification

#### Common SERP Features
| Feature | Description | SEO Action |
|---------|-------------|------------|
| Featured Snippet | Answer box at position 0 | Structure content as Q&A, lists, tables |
| People Also Ask | Expandable questions | Answer related questions in content |
| Local Pack | Map with 3 local businesses | Optimize Google Business Profile |
| Knowledge Panel | Entity information box | Schema markup, Wikipedia, Wikidata |
| Video Carousel | YouTube/video results | Create video content, optimize YouTube |
| Image Pack | Image results | Optimize image alt text, file names |
| AI Overview | AI-generated summary | Create citable, authoritative content |
| News Box | Recent news articles | Publish timely, newsworthy content |
| Reviews | Star ratings in results | Implement Review schema |
| Sitelinks | Sub-page links | Clear site structure, navigation |

### Step 3: AI Answer Analysis

Test keywords in multiple AI systems:

#### ChatGPT/GPT Analysis
```
Query: [Your keyword as a question]
- Does it cite sources?
- Which sources are cited?
- What content type is preferred?
- Is your site/competitor mentioned?
```

#### Claude Analysis
```
Query: [Same keyword]
- Response structure
- Sources mentioned
- How entities are described
```

#### Perplexity Analysis
```
Query: [Same keyword]
- Inline citations
- Source diversity
- Your visibility
```

#### Google AI Overview Analysis
```
Query: [Same keyword on Google]
- Is AI Overview triggered?
- What sources are cited?
- What content format is used?
```

### Step 4: Content Analysis of Top Results

For each top 5 result, analyze:

```
URL:
Word Count:
Content Format: (guide, listicle, how-to, Q&A, etc.)
Heading Structure: (H1, H2 count, keyword usage)
Media: (images, videos, infographics)
Schema Markup: (types implemented)
E-E-A-T Signals: (author, credentials, citations)
Arabic/English: (language ratio)
Mobile Experience: (how it renders)
```

### Step 5: Featured Snippet Opportunities

Snippet types and optimization:

| Type | Format Needed | Best For |
|------|--------------|----------|
| Paragraph | 40-60 word direct answer | Definitions, explanations |
| List | Numbered/bulleted list | How-to, steps, rankings |
| Table | Structured data table | Comparisons, specs, pricing |
| Video | Timestamped video | Tutorials, demonstrations |

```
Snippet Optimization Checklist:
- [ ] Target question directly in H2/H3
- [ ] Provide concise answer immediately after heading
- [ ] Use appropriate format (paragraph, list, table)
- [ ] Include supporting details below
- [ ] Match format of current snippet holder
```

### Step 6: Local SERP Analysis (for Jordan)

- What factors differentiate them?

## Tools for SERP Analysis
- Manual Google searches (use incognito, set location)
- Ahrefs SERP Checker
- SEMrush SERP Features report
- Moz SERP Analysis
- SERPstat
- AI assistants (ChatGPT, Claude, Perplexity)

## Arabic SERP Tips

1. **Use Arabic Google**: google.jo or google.com with Arabic settings
2. **Test different dialects**: Same query in MSA vs Jordanian dialect
3. **Check transliterated terms**: Some users search in English characters
4. **Analyze Arabic competitors**: Different results for Arabic queries
5. **Local modifier impact**: Add "في عمان" vs no location
