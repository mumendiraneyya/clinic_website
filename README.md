# موقع عيادة د. مؤمن ديرانية

![لقطة شاشة للموقع](screenshot.png)

## نظرة عامة

موقع طبي مبني على إطار عمل [Astro](https://astro.build/) مع قالب AstroWind. يدعم اللغة العربية (RTL) ويتضمن:

- صفحة رئيسية تعريفية
- مدونة للمقالات الطبية
- نظام حجز مواعيد عبر Cal.com
- خريطة وموقع العيادة

## التشغيل

```bash
npm install    # تثبيت المتطلبات
npm run dev    # تشغيل خادم التطوير على localhost:4321
npm run claude # يشغل كلاود أن كنت ترغب بالتطوير باستخدام الذكاء الاصطناعي
npm run deploy # بناء ونشر الموقع على abuobaydatajjarrah.pages.dev
```

## تعديل المحتوى

قريباً ستتوفر تعليمات تفصيلية لمساعدة من يرغب في تعديل محتوى الموقع، مع شرح الملفات المطلوب تعديلها لكل قسم.
